Campaign Attribution & ROI Analysis

SQL + Python | Marketing Data Analytics | ROI Modeling

Overview

This project analyzed marketing data across multiple channels (email, paid ads, and social) to understand which campaigns contributed most to conversions. Using SQL and Python, I developed an attribution model to optimize spend and calculate ROI more accurately.

Objectives

  • Identify top-performing campaigns and channels.
  • Develop a weighted attribution model using Python.
  • Provide ROI insights for budget reallocation.

Tools & Skills Used

  • Python (Pandas, Matplotlib, NumPy)
  • SQL (PostgreSQL, Joins, Aggregations)
  • Power BI for Visualization
  • Statistical Modeling & ROI Analysis

Process

Data from different channels were extracted and cleaned. A custom attribution model was created using Python to assign weighted credit to each channel based on conversion paths. Visualizations were built in Power BI to highlight performance insights and ROI by campaign.

Key Outcomes

  • Increased ROI visibility across all marketing channels.
  • Enabled smarter budget decisions that improved conversion rates by 12%.
  • Established a scalable attribution framework for future campaigns.

Collaborators

Ideal collaborators include marketing strategists, data engineers, and campaign managers. Collaboration would enhance channel tagging accuracy and business interpretation of results.
